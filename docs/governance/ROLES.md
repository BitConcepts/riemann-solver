# Roles

## Human researcher

- Final mathematical authority on all proof claims
- Approves any change to benchmark reference data
- Reviews and approves all manuscript changes before submission
- Makes all Clay Prize submission decisions

## AI agents (Oz/specsmith)

- Implement algorithms from peer-reviewed publications
- Run computational verification and falsification experiments
- Draft mathematical exposition for human review
- Propose and execute research iterations
- Generate and maintain governance artifacts

Agent-generated analysis is non-normative until reviewed by the human researcher.
